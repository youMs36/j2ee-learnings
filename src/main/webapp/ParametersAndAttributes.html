Parameters to servlet :

  • - provided by the user in the request
  • - provided by web admin in web.xml
  • - How to get parameters for processing
  • - - String name = request.getParameter("name_Of_parameter");

Attributes to servlet :

  • - used for transferring information in between the Servlet programs.
  • - in one servlet some objects are attached to session, request, config, context, etc
  • - And other servlet can fetch the object to process logic.
  • - How to perform operation with attributes?
    1. - request.setAttribute(String Name, Object value)
    2. - Object value = () request.getAttribtue(String Name)
    3. - request.removeAttribute(String Name)
